1 Srcset y Sizes (Misma imagen, distintos tamaños)

Optimización

El navegador elige la mejor imagen basándose en el ancho de pantalla y la densidad de píxeles (DPI).

¿Cómo funciona?

El atributo srcset proporciona diferentes versiones de la misma imagen con distintos tamaños en píxeles. El atributo sizes define cómo se mostrará la imagen en diferentes breakpoints.

2 Elemento Picture (Art Direction)

Dirección de Arte

Usamos una imagen vertical para móviles y una horizontal panorámica para escritorio.

¿Cómo funciona?

El elemento <picture> permite definir diferentes fuentes de imagen para diferentes contextos (art direction). El navegador selecciona la primera fuente que cumpla con las condiciones.

3 Background Image Responsive (CSS)

CSS Moderno

La imagen de fondo de este div cambia según media queries en CSS.

Contenido sobre el fondo

Este texto es visible sobre la imagen de fondo que cambia según el tamaño de pantalla.

¿Cómo funciona?

Usamos media queries en CSS para cambiar la imagen de fondo según el ancho de la pantalla. Este enfoque es ideal para fondos hero o banners.

4 Comparación de Técnicas

Resumen

srcset + sizes

Ideal para optimizar carga de imágenes con diferentes resoluciones.

  • Optimiza según densidad de píxeles
  • El navegador decide la mejor imagen
  • Misma imagen, diferentes tamaños

Picture Element

Perfecto para dirección de arte (diferentes composiciones según dispositivo).

  • Diferentes recortes o composiciones
  • Ideal para cambiar orientación
  • Formatos modernos (WebP, AVIF)

CSS Background

Mejor para fondos decorativos o imágenes que no son contenido principal.

  • Fácil de implementar con media queries
  • Control total con CSS
  • Ideal para banners y secciones hero
